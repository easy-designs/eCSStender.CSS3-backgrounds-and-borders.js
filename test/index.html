Simple box-shadow

p.test {
  background: #fff;
  box-shadow: 5px 5px 15px 1px #000;
  margin: 2em auto;
  padding: 10px;
  width: 90%;
  }

If your browser supports it, this paragraph should have a drop shadow cast toward it’s lower left, 5px out in each direction. It should be blurred 15px and spread 1px (if supported).

Compound box-shadow

p.test {
  background: #fff;
  box-shadow: 5px 5px 15px 1px #000, -5px -5px 15px 1px #000;
  margin: 2em auto;
  padding: 10px;
  width: 90%;
  }

If your browser supports it, this paragraph should have two drop shadows cast: 1) a drop shadow cast toward it’s lower left, 5px out in each direction, blurred 15px, and spread 1px (if supported)); and 2) a drop shadow cast toward it’s lower left, -5px out in each direction, blurred 15px, and spread 1px (if supported).
